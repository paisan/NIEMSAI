# รายละเอียดโครงการ

## แผนงาน 
การวิจัยและพัฒนานวัตกรรมระบบการแพทย์ฉุกเฉินที่มีมาตรฐานที่ประชาชนเชื่อมั่นและผู้ป่วยฉุกเฉินได้รับบริการอย่างมีคุณภาพ ทั่วถึงและเท่าเทียม ปีงบประมาณ 2569

## ชื่อโครงการ (ภาษาไทย) 
การวิเคราะห์กระบวนงานที่ใช้ปัญญาประดิษฐ์ประยุกต์และแนวปฏิบัติมาตรการป้องกันเพื่อการตัดสินใจในงานบริการด้านการแพทย์ฉุกเฉินที่มีองค์ประกอบปัญญาประดิษฐ์

Services Decision Makings
